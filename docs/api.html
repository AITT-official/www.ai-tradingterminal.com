API Reference

Overview

The AITT API provides programmatic access to trading data, AI analysis, and platform features. This reference documents the available endpoints and how to use them.

Authentication

All API requests require authentication using API keys. You can generate an API key from your account settings. 

Authorization: Bearer YOUR_API_KEY

Base URL

https://api.ai-tradingterminal.com/v1

Market Data

GET Get Market Data

/markets/{symbol}

Retrieves current market data for a specified symbol.

Parameters:

  • symbol (required): Trading pair or asset symbol
  • interval (optional): Time interval (1m, 5m, 1h, 1d)

Example Request:

GET /markets/BTC-USD?interval=1h

Example Response:

{
  "symbol": "BTC-USD",
  "price": 45000.00,
  "change_24h": 2.5,
  "volume_24h": 1250000000,
  "timestamp": "2026-01-06T14:28:09Z"
}

GET Get Historical Data

/markets/{symbol}/history

Retrieves historical market data for analysis.

Parameters:

  • symbol (required): Trading pair or asset symbol
  • from (required): Start timestamp (ISO 8601)
  • to (required): End timestamp (ISO 8601)
  • interval (optional): Data granularity

AI Analysis

POST Request AI Analysis

/analysis/predict

Requests AI-powered market analysis and predictions.

Request Body:

{
  "symbol": "BTC-USD",
  "timeframe": "1h",
  "indicators": ["RSI", "MACD", "EMA"],
  "model": "standard"
}

Response:

{
  "prediction": "bullish",
  "confidence": 0.78,
  "signals": [
    {
      "indicator": "RSI",
      "value": 65,
      "signal": "neutral"
    },
    {
      "indicator": "MACD",
      "value": 120.5,
      "signal": "buy"
    }
  ],
  "timestamp": "2026-01-06T14:28:09Z"
}

Portfolio Management

GET Get Portfolio

/portfolio

Retrieves current portfolio holdings and performance.

Example Response:

{
  "total_value": 100000.00,
  "holdings": [
    {
      "symbol": "BTC",
      "amount": 2.5,
      "value": 112500.00,
      "cost_basis": 100000.00,
      "profit_loss": 12500.00
    }
  ],
  "performance": {
    "day": 2.5,
    "week": 5.2,
    "month": 12.8
  }
}

POST Execute Trade

/trades

Executes a trading order.

Request Body:

{
  "symbol": "BTC-USD",
  "side": "buy",
  "type": "market",
  "amount": 0.5
}

Offline Data

GET Sync Offline Data

/sync

Synchronizes locally cached data with the server.

Parameters:

  • last_sync (required): Last synchronization timestamp
  • entity (optional): Specific entity to sync (trades, portfolio, analysis)

Rate Limits

API requests are rate limited to ensure fair usage:

  • Public endpoints: 100 requests per minute
  • Authenticated endpoints: 500 requests per minute
  • Trading endpoints: 50 requests per minute

Error Handling

The API uses standard HTTP status codes and returns error details in JSON format: 

{
  "error": {
    "code": "INVALID_SYMBOL",
    "message": "The specified symbol is not supported",
    "details": "BTC-USDT is not available on this exchange"
  }
}

Common Error Codes:

  • 400 - Bad Request: Invalid parameters
  • 401 - Unauthorized: Invalid or missing API key
  • 403 - Forbidden: Insufficient permissions
  • 404 - Not Found: Resource not found
  • 429 - Too Many Requests: Rate limit exceeded
  • 500 - Internal Server Error: Server error

WebSocket API

For real-time data, use the WebSocket API: 

wss://api.ai-tradingterminal.com/ws

Subscribe to Market Updates:

{
  "action": "subscribe",
  "channel": "market",
  "symbol": "BTC-USD"
}

SDK Libraries

Official SDKs are available for popular programming languages:

  • JavaScript/TypeScript: npm install @aitt/sdk
  • Python: pip install aitt-sdk
  • Go: go get github.com/AITT-official/aitt-go
